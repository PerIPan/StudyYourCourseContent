Key Design Decisions

  • OpenAI for embeddings only — text-embedding-3-small is cheap ($0.02/1M tokens), best-in-class for vector search. Claude for the actual reasoning.
  • Chunk size: ~500 tokens with 50-token overlap — good balance for lecture material (slides vs dense papers)
  • Source references — each chunk stores course name, lecture number, page/slide number. Answers cite these.
  • OpenWhisperer integration — new endpoint in unified_server.py that calls /api/chat, pipes response through TTS
